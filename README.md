# athorrent-backend
A lightweight seedbox backend

## prerequisites
- [boost](http://www.boost.org/)
- [libtorrent](http://www.libtorrent.org/)
- [rapidjson](http://rapidjson.org/)

## building
``` sh
./bootstrap.sh
./configure
make
```
